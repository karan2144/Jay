# Render Full Production Telegram Scalping Bot

## Features
- Real-time Telegram listener
- Anti-sleep Flask server
- Auto reconnect on crash
- Flood protection
- Environment variable safety check

## Render Setup

Build Command:
pip install -r requirements.txt

Start Command:
python main.py

## Environment Variables (Add in Render Dashboard)

API_ID = your_api_id
API_HASH = your_api_hash
SOURCE_ID = -100xxxxxxxxxx
TARGET_ID = -100xxxxxxxxxx
